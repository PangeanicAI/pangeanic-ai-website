## Overview

Sundanese AI systems require more than generic multilingual corpora. Enterprise-grade regional AI applications must understand dialect diversity, conversational Sundanese, Indonesian-Sundanese code-switching behavior, culturally grounded terminology and real-world communication patterns across West Java and broader Indonesian ecosystems.

This repository documents Pangeanic’s Sundanese AI data capabilities for multilingual AI systems, Southeast Asian AI initiatives, speech technologies, machine translation, enterprise NLP and multimodal AI applications.
